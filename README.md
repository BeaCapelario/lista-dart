Este repositório apresenta o desenvolvimento de uma aplicação de lista utilizando Dart e Flutter. O objetivo principal foi explorar a manipulação de coleções de dados e a renderização dinâmica de elementos em uma interface mobile, focando em usabilidade e organização de código.

---

### O que foi explorado:
* Listas Dinâmicas: Implementação de ListView para exibição de múltiplos itens de forma eficiente.
* Manipulação de Coleções: Uso de métodos da linguagem Dart para adicionar, remover e listar dados.
* Widgets Reutilizáveis: Criação de componentes de interface para padronização visual da lista.
* Entrada de Dados: Uso de campos de texto e controladores para capturar informações do usuário.
* Feedback Visual: Atualização da interface conforme a modificação dos dados na lista.
